projects

Adapting Language Models for Low-Resource GPU Kernel Programming
  • Teaching LLMs to write fast, compiler-verified GPU kernels for AMD HIP, a low-resource language where open-source training data is scarce.
  • Built a synthetic dataset of 400 PyTorch tasks and a multi-agent pipeline that translates and optimizes kernels via evolutionary search with MI350X benchmarking.
  • Trained with SFT then GRPO reinforcement learning using hardware-in-the-loop rewards, evaluated on KernelBench for compilation, correctness, and speedup over PyTorch.
▶ Presented at AMD AI DevDay 2026! →
Egocentric Future Action Anticipation with Plan-and-Verify Reinforcement Learning
  • Predicts what a person will do next from first-person video, the anticipation embodied AI in robots, AR/VR, and smart glasses depends on.
  • Built on a Qwen2-VL vision-language model over the Ego-Exo4D dataset, comparing supervised fine-tuning (LoRA) against GRPO reinforcement learning.
  • Uses a plan-and-verify reward: the model drafts a first-person plan, then an exocentric verifier checks it from a third-person view, predicting the next 30 seconds of activity against ground-truth keysteps.

CodeCrack: a voice-driven AI agent simulating dynamic, real-time mock interviews
  • A voice-driven mock interview platform that recreates the pressure of a live coding interview, with an ElevenLabs voice agent playing a senior software engineer across both behavioral and technical rounds.
  • Built on Vercel with an in-browser IDE (Monaco editor plus a Pyodide Python interpreter) so candidates write and run real code, with LeetCode-style questions pulled from a HuggingFace dataset.
  • An agentic system tracks the full interaction state, streaming difficulty-matched questions for the agent to read aloud, testing submissions on "Run Code," offering open-ended hints, and returning final feedback on communication, problem-solving, and debugging via ElevenLabs' criteria evaluation.
★ 1st Place, Best Use of ElevenLabs at Stanford TreeHacks

A Search & Rescue Agent for Hazardous Grid Environments Using Reinforcement Learning
  • Trains an autonomous agent to run search-and-rescue missions in a grid world, dodging moving fire hazards to reach a trapped victim and a safe exit.
  • A Proximal Policy Optimization (PPO) agent learns the task under stochastic fire movement and sparse rewards.
  • The resulting policy reliably avoids hazards and completes the full objective sequence through coordinated obstacle avoidance and long-horizon planning.
